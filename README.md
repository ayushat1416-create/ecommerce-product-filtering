# E-Commerce Product Filtering API

Full-stack mini project built during internship.

## Tech Stack

Backend:
- Node.js
- Express.js
- MongoDB (Mongoose)

Frontend:
- React (Vite)

## Features

- Filter by Category
- Filter by Price Range
- Sort by Price (Low → High / High → Low)
- Server-side Pagination
- Optimized MongoDB queries with indexes

## Installation

### Backend
cd backend
npm install
npm run seed
npm run dev

### Frontend
cd frontend
npm install
npm run dev

Backend runs on:
http://localhost:5000

Frontend runs on:
http://localhost:5173
